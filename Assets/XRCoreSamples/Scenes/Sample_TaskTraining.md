# Sample_TaskTraining

Objetivo:

- Validar `XRCore.Tasks` con `XRCore.Interaction` via `XRCoreEventBus`.

Setup minimo recomendado:

1. Crear escena `Sample_TaskTraining`.
2. Anadir `XRTaskRunner` con un `XRTaskDefinition` de ejemplo.
3. Anadir uno o mas `XRInteractionSignalEmitter` y conectar `RaiseSignal`.
4. Verificar que los pasos automaticos avanzan con eventos de senal.
