XRCore is a modular product line for Unity XR teams that need guided workflows, measurable outcomes, and scalable authoring.

Core architecture:

`Perception -> Events -> Reasoning -> Action -> Context -> Analytics`

## Product Portfolio

### Foundation
- **XRCore SDK**  
  Event-driven runtime foundation for XR assistants.  
  [Repository](https://github.com/splibiplay/xrcore-sdk) · [Unity Asset Store](https://assetstore.unity.com/packages/tools/ai-ml-integration/xrcore-event-driven-ai-framework-for-unity-xr-366852)

- **XRCore Context**  
  Shared runtime memory and context contracts for all modules.  
  [Repository](https://github.com/splibiplay/xrcore-context)

### Training Stack
- **XRCore Training Toolkit**  
  Guided scenario runtime, validators, and training UX.  
  [Repository](https://github.com/splibiplay/xrcore-training-toolkit)

- **XRCore Training Assessment**  
  Scoring, pass/fail, critical failures, and reports.  
  [Repository](https://github.com/splibiplay/xrcore-assessment)

- **XRCore Training Authoring**  
  Editor-first scenario authoring and generation tools.  
  [Repository](https://github.com/splibiplay/xrcore-training-authoring)

### Intelligence Extensions
- **XRCore Voice**  
  Voice intent and command layer for guided flows.  
  [Repository](https://github.com/splibiplay/xrcore-voice)

- **XRCore VisionPlus**  
  Vision-driven scene intelligence and perception hooks.  
  [Repository](https://github.com/splibiplay/xrcore-visionplus)

- **XRCore LLBridge**  
  Bridge layer between XR events and LLM orchestration.  
  [Repository](https://github.com/splibiplay/xrcore-llbridge)

- **XRCore Analytics**  
  Cross-module telemetry and operational KPIs.  
  [Repository](https://github.com/splibiplay/xrcore-analytics)
