# SPL XR - XRCore Ecosystem

Official XRCore product suite for Unity XR.

## XRCore Ecosystem

- **XRCore SDK**  
  Modular AI agent framework for Unity XR (event-driven architecture).

- **XRCore Training Suite**  
  Professional training products built on XRCore.
